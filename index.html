Welcome to Digitivia AI Agent! This platform allows you to create, manage, and monitor powerful AI agents across multiple communication channels.

ðŸ¤– Agent Roles & System Prompts

The System Prompt is the brain of your agent. It defines who the agent is, what it knows, and how it speaks.

  • Role: Define the persona (e.g., Support Agent, Sales Rep).
  • Tone: Adjectives that describe the personality (e.g., Professional, Friendly, Helpful).
  • Knowledge Base Links: Add URLs that your agent should use as a reference to answer questions accurately.
Pro Tip: Be specific in your System Prompt. Instead of "You are a helpful assistant," try "You are an expert real estate consultant for Acme Realty, specializing in downtown apartments."
